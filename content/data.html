Welcome to the Drosera Database.

The Drosera Database is a collection of information about programming languages. If you see anything that is familiar to you, then click on it and you will see a collection of similar interfaces found in other languages. This is the multi-language programming database.

This database is separated into multiple sections:

Languages
where you can browse concepts by programming language,
e.g.: C, C++, C#, Java, Python, Ruby, etc.
Dictionaries
where you can browse concepts by OpenMath/MathML content dictionary names, e.g.: arith1, logic1, relation1, etc.
Namespaces
where you can browse concepts by XML namespace, or QNames,
e.g.: HTML, SVG, DocBook, etc.
Vocabularies
where you can browse concepts by RDF vocabularies, or URIs,
e.g.: RDF, RDFS, OWL, SKOS, etc.
